Возможности

Всё для управления фермой в одном месте

Мониторинг на уровне железа

Хешрейт, температура, потребление, эффективность (J/TH), аптайм и прошивка — телеметрия прямо с устройств, а не только из API пула.

Иерархия площадок

Площадка → контейнер → стойка → ASIC. Наглядная навигация, в которой не теряешься даже на тысячах воркеров.

Сканер сети

Автопоиск ASIC по IP-диапазону: тип, модель, пул, хешрейт — и добавление в систему в один клик. Никакого ручного ввода сотен адресов.

Тепловые карты стоек

Вид стойки «сетка / полки / тепло». Перегрев виден на схеме раньше, чем воркер отвалится.

Алерты, где удобно

Уведомления в Telegram, на почту и в другие мессенджеры. Порог сработал — вы уже знаете, не дожидаясь простоя.

Данные остаются у вас

Ставится на ваш сервер и работает в изолированной сети. Размер, локация и хешрейт фермы не уходят наружу.

Сканер сети

Развернуть ферму на сотни машин — за минуты

Задайте IP-диапазон и порты — RackGrid найдёт все ASIC, определит тип и модель, покажет статус, пул и хешрейт. Останется отметить нужные и добавить в систему.

  • Автоопределение Antminer, Whatsminer, Vnish и других.
  • Настраиваемые порты, таймаут и до 128 потоков сканирования.
  • Добавление найденного оборудования в один клик.

Иерархия и стойки

От площадки целиком до конкретного слота

Оборудование сгруппировано так, как ферма устроена физически. По каждому ASIC — хешрейт, температура, мощность, аптайм, прошивка, IP и слот; действия «в сервис», «привязать владельца», «удалить».

  • Вид стойки сетка / полки / тепло с цветовым статусом слотов.
  • Статусы онлайн / предупреждение / ошибка / нет связи.
  • Быстрый поиск по ASIC, стойке и контейнеру.
Для хостинга оборудования

Каждый клиент видит только свои ASIC

Роли и привязка владельцев делают RackGrid готовым для хостинг-площадок: разграничение доступа по площадкам, отдельный кабинет клиента без лишнего, полный контроль у администратора.

админСупер-админ. Полный доступ, управление пользователями и лицензией.

клиентПользователь. Видит только привязанные к нему ASIC — идеально для клиента хостинга.
Лицензирование

Лицензия под размер фермы — по устройствам и сроку

Лицензия ограничена количеством ASIC и сроком действия, привязана к железу вашего сервера. Прозрачно видно использование квоты и дату окончания. Гибко: свернули часть машин — не продлеваете на простаивающие. Есть пробный период.

  • Квота по числу устройств и сроку — под текущий размер фермы.
  • Привязка к аппаратному отпечатку сервера, офлайн-активация .lic-файлом.
  • Пробный период для теста на вашем железе перед покупкой.

Почему свой сервер — это не «сложнее», а безопаснее

RackGrid · свой сервер

ваш сервер · ваши данные
  • Данные фермы не покидают периметр — размер, локация, хешрейт скрыты от третьих лиц.
  • Работает в изолированной сети, за NAT/VPN, без выхода в интернет.
  • Несколько площадок через распределённых агентов.
  • Дорабатывается под вашу инфраструктуру — свои дашборды, интеграции, автоматика.

Облачные сервисы

чужой сервер · чужие правила
  • Телеметрия всей фермы хранится на стороне провайдера.
  • Нужен постоянный доступ в интернет с каждого устройства.
  • Сбой облака = слепая зона по всей ферме.
  • Только штатный функционал платформы — кастома нет.
